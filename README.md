# React Native Map Application с Геолокацией и Уведомлениями – Задание 3
Продолжая работу над предыдущими заданиями, в этом задании вам предстоит добавить GPS-трекинг и локальные уведомления. Теперь приложение будет отображать текущее местоположение пользователя и уведомлять его при приближении к сохранённым меткам.
# Инструкции по установке

1. Клонируйте репозиторий:
```
   git clone ...
   cd ...
```
Или скачайте ZIP архив:
- Зайдите на GitHub репозиторий
- Нажмите "Code" → "Download ZIP"
- Распакуйте архив
2. Установите зависимости:
```
   npm install --legacy-peer-deps

```
   
3. Запустите проект через Expo:
```
 npm expo start
```
   
4. Откройте приложение в Expo Go на вашем устройстве.
Отсканируйте QR-код в терминале:
- Android: Откройте Expo Go → "Scan QR Code"
- iOS: Отсканируйте камерой, откроется Expo Go

  # Инструкция по тестированию

1. Запустите приложение через Developer Build
2. При запуске приложение запросит разрешение на доступ к местоположению и уведомлениям.
3. На карте отображается текущее местоположение с синим маркером.
4. Добавьте маркеры долгим нажатием на карту.
5. Для тестирования функционала геолокации можно воспользоваться фейковым GPS в режиме разработчика (на платформе Android).
6. Для тестирования функионала уведомлений можно также использовать фейковый GPS, перемещая фейковую метку геолокации ближе и дальше от конкретных маркеров (предел расстояния для появления уведомлений - 500м).

# Описание стратегии управления уведомлениями

- Используется порог в 500 метров для определения приближения к метке.
- Активные уведомления хранятся в памяти, чтобы не допускать дублирование уведомлений для одной метки.
- При удалении пользователя из зоны метки активное уведомление для этой метки очищается, позволяя уведомлять повторно при повторном приближении.
- Все уведомления локальные, реализованы с помощью expo-notifications.

# Известные ограничения

- В некоторых случаях периодическое обновление местоположения может иметь задержки из-за политики энергосбережения ОС.
- На старых устройствах может быть ограничена точность геолокации.
